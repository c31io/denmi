# 向量空间

域上的模。

## 定义

\newcommand{\u}{\mathbf u}
\newcommand{\v}{\mathbf v}
\newcommand{\w}{\mathbf w}
\newcommand{\0}{\mathbf 0}

| 公理 | 陈述 |
| - | - |
| 加法结合律 | $\u+(\v+\w)=(\u+\v)+\w$ |
| 加法交换律 | $\u+\v=\v+\u$ |
| 加法单位元 | $\exists\0\in V,\forall\v\in V,\v+\0=\v$ |
| 加法逆元 | $\forall\v\in V,\exists-\v\in V,\v+(-\v)=\0$ |
| 乘法兼容律 | $a(b\v)=(ab)\v$ |
| 乘法单位元 | $1\v=\v$ |
| 向量加分配律 | $a(\u+\v)=a\u+a\v$ |
| 域加分配律 | $(a+b)\v=a\v+b\v$ |
